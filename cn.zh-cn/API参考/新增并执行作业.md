# 新增并执行作业<a name="ZH-CN_TOPIC_0176713997"></a>

## 功能介绍<a name="section4408504619327"></a>

在MRS集群中新增并提交一个作业。

>![](public_sys-resources/icon-note.gif) **说明：**   
>-   如需使用OBS加密功能，请先参考[使用OBS加密数据运行作业](https://support.huaweicloud.com/usermanual-mrs/mrs_01_0640.html)页面进行相关配置后，再调用API接口运行作业。  
>-   当集群开启Kerberos认证时，需要先在集群详情页的“概览”页签，单击“IAM用户同步“右侧的“点击同步”进行IAM用户同步，然后再通过该接口提交作业。

## 示例<a name="section1210015461189"></a>

-   请求示例

    MapReduce作业请求示例：

    ```
    {
        "job_name":"MapReduceTest",
        "job_type":"MapReduce",
        "arguments":[
            "obs://obs-test/program/hadoop-mapreduce-examples-x.x.x.jar",
            "wordcount",
            "obs://obs-test/input/",
            "obs://obs-test/job/mapreduce/output"
        ],
        "properties":{
            "fs.obs.endpoint":"obs endpoint",
            "fs.obs.access.key":"xxx",
            "fs.obs.secret.key":"yyy"
        }
    }
    ```

    SparkSubmit作业请求示例：

    ```
    {
        "job_name":"SparkJobTest",
        "job_type":"SparkSubmit",
        "arguments":[
            "--master",
            "yarn",
            "--deploy-mode",
            "cluster",
            "--py-files",
            "obs://obs-test/a.py",
            "--conf",
            "spark.yarn.appMasterEnv.PYTHONPATH=/tmp:$PYTHONPATH",
            "--conf",
            "spark.yarn.appMasterEnv.aaa=aaaa",
            "--conf",
            "spark.executorEnv.aaa=executoraaa",
            "--properties-file",
            "obs://obs-test/test-spark.conf",
            "obs://obs-test/pi.py",
            "100000"
        ],
        "properties":{
            "fs.obs.access.key":"xxx",
            "fs.obs.secret.key":"yyy"
        }
    }
    ```

    HiveScript作业请求示例：

