-   请求示例

    ```
    {
    	"job_id_list": ["c23c1f53-5c8e-4eb8-ab2f-a6acff8ac369", "8f7969b6-d2fb-4442-9533-3fe7d7bdf31b"]
    }
    ```

-   响应示例
    -   成功示例

        无

    -   失败示例

        ```
        {
        "error_msg":"批量删除作业失败",
        "error_code":"0161"
        }
        ```
