# 配置弹性伸缩规则<a name="AutoScalingPolicy"></a>

## 功能介绍<a name="section1693101712257"></a>

对弹性伸缩规则进行编辑。

在创建集群并执行作业接口中也可以创建弹性伸缩规则。

## 接口约束<a name="section14947917122516"></a>

无

## URI<a name="section1595791782513"></a>
